## 线程的状态和生命周期
java使用Thread类及其子类对象表示线程，新建线程在它的一个完整生命周期内通常经历**5种状态**，通过控制和调度使这几种状态转化；
1. **新建状态**(Newborn)
2. **就绪状态**(Runnable)
- CPU给时间片
3. **运行状态**(Running)
4. **阻塞状态**(Blocked)
- 时间片被抢走
5. **消亡状态**(Dead)
- 运行结束后即消亡
